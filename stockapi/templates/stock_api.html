Stock Data API

To get stock price information on a given stock on a single date, use the following url:

/stock_price_details/?code={STOCK_CODE}&date={YYYY-MM-DD}

For Example:

/stock_price_details/?code=EGAD&date=2022-08-03

To get stock price information on a given stock between two dates, use the following url:

/get_stock_information/?code={STOCK_CODE}&start_date={YYYY-MM-DD}&end_date={YYYY-MM-DD}

For Example:

/get_stock_information/?code=EGAD&start_date=2022-05-03&end_date=2022-06-03
